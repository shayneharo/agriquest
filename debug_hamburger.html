This page forces the hamburger menu to be visible for testing.

What to look for:
  • Hamburger Icon: Three white horizontal lines in the top right
  • Click Animation: Lines should rotate to form an X when clicked
  • Menu Expansion: Navigation should slide down when clicked
  • Debug Info: Check the top-right corner for viewport info
If you don't see the hamburger:
  1. Check if you're on a mobile device or narrow browser window
  2. Try refreshing the page (Ctrl+F5 or Cmd+Shift+R)
  3. Check browser console for any JavaScript errors
  4. Try opening in an incognito/private window
